# citylife
基于flask的同城本地信息平台，毕业设计。
本项目支持布局自适应移动设备，但自适应是在bootstrap3的时候适配的，后期项目改用bootstrap4部分页面或功能未做自适应适配。
注册采用邮箱验证方式，杜绝机器人。
